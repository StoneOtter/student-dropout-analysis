# student-dropout-analysis

Project Instructions
This repository contains the dataset and the R code used for our analysis

Dataset
The dataset is uploaded as a ".zip file"
It was compressed to prevent possible file corruption when uploading

Before running the code:
1. Download the .zip file
2. Extract the dataset on your computer
3. Copy the "file location (file path)" of the extracted dataset

Code Setup
Inside the R code, the "file path for the dataset is written at the top of the first code block".
After extracting the dataset, you must "change the file path in the code to match the location on your computer".

Example:
Current path in the code:
```data <- read.csv("C:/Users/Example/Downloads/dataset.csv")```

You must change it to your own path, for example:
```data <- read.csv("C:/Users/YourName/Documents/dataset.csv")```


Important
Make sure the dataset is "extracted before running the code"
Update the "file path correctly for it to work", otherwise the code will not load the dataset
After making changes to the code, "commit the updated version so the group can see the changes"(that are beneficial but the code is pretty much done)
